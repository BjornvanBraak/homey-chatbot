Homey Chatbot

We wanted to showcase the feasibility of making a chatbot. This chatbot uses both preprogrammed logic and generative AI for providing answers. It does this by first looking if the question matches any preprogrammed logic. If nothing was found the generative AI will look through to both https://homey.app/en-us/ and https://community.homey.app/ for revelant answers. Furthermore, at any time the user can request the conversation to be escalate to an agent so they are not locked into using the bot.

Suggestions for asking the chatbot

How can I buy homey online? (Programmed Logic)

Can I use matter with my homey? And how does that work? (Knowledge Base)

My Homey Pro 2019 edition has trouble connecting with Wifi? (Community Forum)

Can I talk to an agent instead of the bot? (Escalate to agent)

Okay, bye (Collect feedback at end of conversation)

Please feel free to try out your own ideas!!!
